# 🐧 Linux学習記録 (linux-study)

高校1年生からSRE/DevOps、クラウドアーキテクト設計職を目指し、Linuxの基本操作からシステム運用・自動化まで、毎日インプットとアウトプットしていくためのリポジトリです。
---
## 学習の目的
* Linuxの操作（ターミナル操作）に完全に慣れる
* 基本的なコマンドを使いこなし、システムの仕組みを理解する
* 学んだことをドキュメントとして整理し、アウトプット力を高める
* 継続する力を身につける
---
## 学習環境
- MacBook Air（Apple Silicon）
- Docker Desktop
- Ubuntu（Dockerコンテナ環境）
- VSCode
- Git / GitHub
---
## 現在取り組んでいること
- Linuxコマンドの理解・実践
- シェルスクリプトによる処理の自動化
- cronによる定期実行設定
- ログ解析・システム状態確認
- Dockerを使用した環境構築
- バックアップ処理の自動化
