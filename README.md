# 🐧 Linux学習記録 (linux-study)
高校1年生からSRE/DevOps、クラウドアーキテクト設計職を目指し、Linuxの基本操作毎日インプット・アウトプットしていくためのリポジトリです。

* Linuxの操作（ターミナル操作）に完全に慣れる
* 基本的なコマンドを使いこなし、システムの仕組みを理解する
