# Linuc学習 Day4
## 覚えたコマンド
- ls 一覧を表示
- ls -l 詳細付きで一覧表示
- ls -a 隠しファイル含めて全部表示
- ls -la ls -lとls -aの合体版。隠しファイル含めて詳細付きで全部表示
- cat ファイル名 ファイル名 で同時に違うファイルが見れる
- mkdir フォルダ名 フォルダ名 で同時に違うフォルダを一気に作成
- find 探す
- . 今いる場所(から)
- .. 一個上の場所
- -name　名前で検索(findのオプション)
- *はなんでもok。linuxで使えるワイルドカード
- "*.txt" は最後が.txtならなんでもok
- | 左の結果を右に渡す
- wc 行数、単語数、文字数を数える
- wc -l 行数だけを数える
- wc -w 単語数だけを数える
- wc -c バイト数(文字数)だけを数える
## 間違えたこと
- cat ren1 renn2 がエラーになった。
- 原因 renn1とrenn2はフォルダ名でファイルではなかった
- find がdpoute になった。
- 原因 "を閉じ忘れた
- 解決 control+cでキャンセルしてやり直した
## 学んだこと
- catはファイルの内容を表示するコマンド
- "は必ず閉じる
- 間違えた時はcontrol+cでキャンセルすればやり直せる
